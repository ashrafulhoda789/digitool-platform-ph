# Title: DigiTools Platform

Digital Tools Buying Platform is a modern and responsive web application built with React that allows users to explore and purchase various digital services and tools such as Basic plans, Standard plans, and Premium plans etc.

--- 
## Technology used:
- HTML5, Tailwind CSS, DaisyUI
- JavaScript (ES6+)
- React.js

## Project Features:
- User login
- User can explore and purchase services
- User can add items to cart which will dynamically show in cart button and navbar
- User can remove items from cart and purchase items.
